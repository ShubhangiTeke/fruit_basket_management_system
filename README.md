#  Fruit Basket Management System

A menu-driven Java console application that demonstrates Object-Oriented Programming (OOP) concepts such as abstraction, inheritance, polymorphism, encapsulation, and method overriding.

---

## Features

- Add Apple, Mango, and Orange to the basket
- Display all fruit names
- Display fresh fruits with their details and taste
- Display the taste of stale fruits
- Mark a fruit as stale using its index
- Mark all sour fruits as stale

##  How to Run

1. Clone this repository.
2. Open the project in Eclipse, IntelliJ IDEA, or VS Code.
3. Compile the Java file.
4. Run the `Fruitbasket` class.
5. Enter the basket size.
6. Use the menu to perform different operations.

---

##  Menu Options

```
0. Exit
1. Add Mango
2. Add Orange
3. Add Apple
4. Display Fruit Names
5. Display Fresh Fruits
6. Display Taste of Stale Fruits
7. Mark Fruit as Stale
8. Mark All Sour Fruits as Stale
```
