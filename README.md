# Fruit Basket Management System

A menu-driven Java console application demonstrating Object-Oriented Programming concepts.

## Features

- Add Apple, Mango, and Orange
- Display all fruit names
- Display fresh fruits with details
- Display taste of stale fruits
- Mark a fruit as stale
- Mark all sour fruits as stale

## Concepts Used

- Abstract Class
- Inheritance
- Polymorphism
- Encapsulation
- Method Overriding
- Arrays
- Scanner Class
- Menu Driven Programming

## Technologies

- Java
- Eclipse / VS Code
